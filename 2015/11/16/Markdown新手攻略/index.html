Markdown新手攻略

发表于 Nov 16 2015   |   分类于 Markdown   |     |   阅读次数   |  

Markdown 是一门极简的轻量标记语言，易读易写易用，兼容性好，极其适用网络书写。它的精髓在于能让文档更容易读、写和随意改，同时语法简洁明了，容易掌握。本文以示例方式介绍其基本语法。
